# AutoForce Content Portal

Portal público para organizar entregas semanais de conteúdo da AutoForce.

## Estrutura
- `index.html` — home do portal
- `2026/05/semana-2026-05-17/` — primeira semana publicada

## Organização padrão
- `YYYY/MM/semana-YYYY-MM-DD/`

Cada pasta semanal pode conter:
- `index.html`
- peças da semana
- legendas
- PDFs/README
- materiais de revisão
